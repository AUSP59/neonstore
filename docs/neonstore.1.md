% NEONSTORE(1) | NeonStoreSystem Manual
# NAME
neonstore - store management CLI

# SYNOPSIS
**neonstore** *command* [**--options**]

# DESCRIPTION
Manage products and orders with CSV or SQLite backends.

# COMMANDS
*add-product*, *list-products*, *restock*, *sell*, *save*, *load*

# OPTIONS
**--backend**, **--dir**, **--dsn**, **--config**, **--output**, **--no-color**, **--verbose**

# EXIT STATUS
0 on success; 1 usage; 2 runtime error; 3 backend unavailable.
