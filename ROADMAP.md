# Roadmap

## v1.1 (Near-term)
- Structured **CPack** packages (.deb/.rpm/.zip) with checksums.
- **CodeQL** analysis and release automation.
- Additional CSV validations and import diagnostics.

## v1.2
- Optional **SQLite** backend (enabled only if dependency is available).
- Basic **Prometheus-style** metrics (opt-in) for the CLI.

## v2.0
- Pluggable storage interface (CSV / SQLite / REST adapter).
- Multi-user roles and authentication (**only** via vetted libraries).

*This roadmap is informative, not a contract. See `CONTRIBUTING.md` for how to propose changes.*
